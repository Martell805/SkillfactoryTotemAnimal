# 🐾 Бот «Тотемное животное» — Московский зоопарк

Telegram-бот-викторина для популяризации программы опеки Московского зоопарка.

## 📋 Описание

Бот помогает пользователю найти его «тотемное животное» среди обитателей Московского зоопарка. После прохождения викторины пользователь получает результат с фотографией животного и информацию о программе опеки «Клуб друзей».

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/your-username/moscow-zoo-bot.git
cd moscow-zoo-bot
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка токена

Перед запуском создайте переменную среды BOT_TOKEN, содержащую токен бота

```bash
export BOT_TOKEN=1234567890:ABCDefghijklmnopqrstuvwxyz  # Linux/Mac
set BOT_TOKEN=1234567890:ABCDefghijklmnopqrstuvwxyz     # Windows
```

### 5. Запуск бота
```bash
python main.py
```

## ⚙️ Технические детали

### Алгоритм подбора животного
Каждый вариант ответа начисляет очки одному или нескольким животным. После ответа на все 8 вопросов бот суммирует очки и выбирает победителя. Система взвешивания позволяет учитывать тонкие различия в характере пользователя.

### Команды бота
| Команда    | Описание                   |
|------------|----------------------------|
| `/start`   | Главное меню и приветствие |
| `/quiz`    | Сразу начать викторину     |
| `/about`   | О программе опеки          |
| `/contact` | Контакты зоопарка          |
| `/help`    | Справка                    |

### Функциональность
- ✅ 8 вопросов с 4 вариантами ответа
- ✅ Взвешенный алгоритм подсчёта очков
- ✅ Результат с фото животного
- ✅ Информация о программе опеки с ценой
- ✅ Кнопка «Поделиться» (с упоминанием бота)
- ✅ Контакт с зоопарком (email + телефон)
- ✅ Перезапуск викторины
- ✅ Система оценки (5 звёзд)
- ✅ Логирование ошибок и событий
- ✅ Асинхронная обработка (python-telegram-bot v20)

## 🔒 Безопасность и масштабируемость

- Токен бота хранится в переменной окружения (не в коде)
- Защита от повторного нажатия кнопок ответа
- Асинхронная архитектура выдерживает множество одновременных пользователей
- Все ошибки логируются в `bot.log`

## 📈 Мониторинг

Бот ведёт два вида логов:
- **Консоль** — для оперативного мониторинга при запуске
- **bot.log** — постоянный файл для анализа ошибок

Отзывы пользователей (оценки от 1 до 5 звёзд) сохраняются в памяти. Для продакшна рекомендуется подключить SQLite или PostgreSQL.

## 🛠️ Расширение и кастомизация

### Добавить новое животное
1. Добавьте запись в `ANIMALS` в `data/quiz_data.py`
2. Добавьте URL фото в `ANIMAL_IMAGES` в `handlers/bot_handlers.py`
3. Добавьте очки нового животного в нужные варианты ответов в `QUESTIONS`

### Добавить новый вопрос
Добавьте новый словарь в список `QUESTIONS` в `data/quiz_data.py`, соблюдая структуру:
```python
{
    "id": 9,
    "text": "Текст вопроса?",
    "options": [
        {"text": "Вариант 1", "scores": {"волк": 2, "лев": 1}},
        ...
    ],
}
```
